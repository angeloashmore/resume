# angeloashmore.github.io

Personal website for Angelo Ashmore.

## Build Instructions

1. Install gem dependencies:

   ```shell
   bundle install
    ```

1. Serve locally at [http://localhost:4567](http://localhost:4567):

   ```shell
   bundle exec middleman server
   ```

   or build to `/build`:

   ```shell
   bundle exec middleman build
   ```
